## Модель движения Human
Каждый объект **Human** на каждом шаге времени делает случайный шаг.  

Формулы:  
x(t+1) = x(t) + v * cos(θ)  
y(t+1) = y(t) + v * sin(θ)  

Где:  
- v — скорость (расстояние за шаг),  
- θ — случайный угол от 0 до 2π.  

Таким образом, на каждом шаге выбирается новое направление, и траектория получается как случайное блуждание на плоскости.
