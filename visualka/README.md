## Задание №3: Интерфейсы и Рефакторинг

### Структура проекта (1 файл = 1 класс/интерфейс):

| Файл | Описание |
| :--- | :--- |
| **`Movable.kt`** | **Интерфейс.** Определяет контракт (метод `fun move()`) для всех объектов, способных к движению, в рамках ООП-модели. |
| **`Human.kt`** | **Класс.** Базовый класс, реализует интерфейс `Movable` (движение по модели Random Walk). Класс сделан `open` для наследования. |
| **`Driver.kt`** | **Класс.** Наследник `Human`. Переопределяет метод `move()` для реализации **прямолинейного движения**. |
| **`Main.kt`** | **Функция `main()`**. Точка входа в программу. Создает и параллельно запускает объекты `Human` и `Driver`. |
